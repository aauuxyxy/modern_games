## API 一覧

| No  | メソッド | パス                                           | 用途                             |
| --- | -------- | ---------------------------------------------- | -------------------------------- |
| 1   | GET      | /cat_games?arg1={categoryId}&arg2={categoryId} | カテゴリー別のゲームデータを取得 |
| 2   | GET      | /search_games?arg1={searchWord}                | ゲームデータのワード検索用 API   |
| 3   | GET      | /categorys                                     | カテゴリー一覧を取得             |
